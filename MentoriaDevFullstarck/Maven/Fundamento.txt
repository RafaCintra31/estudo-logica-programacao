MAVEN (apache)
-----

- Cuidar do classpath
- Gerenciar as dependencias de projeto (libs externas)
- Determina a estrutura do projeto

- Config maven
    -> pom.xml (fica na raiz do projeto, é o coração)
    -> .m2     (diretorio onde ficam as libs externas)
    -> target  (diretorio em que fica nosso codigo compilado)

Comandos (goals)
    - mvn clean   (Limpa a target)
    - mvn compile (Compila o codigo)
    - mvn package (Gera o jar na target)

Estrutura pasta
    - SRC
        - MAIN
            - JAVA
            - RESOURCES
        - TEST   
            - JAVA
            - RESOURCES
